# 📊 Netflix Data Analysis Project

## 📌 Overview

This project focuses on analyzing Netflix's dataset to extract meaningful insights using Python.
The analysis explores trends in content type, release years, countries, genres, and more.

The goal of this project is to demonstrate data cleaning, manipulation, visualization, and storytelling skills using real-world data.

---

## 🎯 Objectives

* Clean and preprocess raw dataset
* Perform exploratory data analysis (EDA)
* Visualize trends and patterns
* Derive actionable insights

---

## 🛠️ Technologies Used

* Python 
* Pandas
* Matplotlib
* Google Colab

---

## 📂 Dataset

* Source: Netflix dataset (CSV file)
* Contains information such as:

  * Title
  * Director
  * Cast
  * Country
  * Release Year
  * Rating
  * Duration
  * Genre (Listed In)

---

## 🧹 Data Cleaning Steps

* Handled missing values
* Removed duplicates
* Converted data types
* Standardized column formats

---

## 📊 Analysis & Visualizations

Some key analyses performed:

* Distribution of Movies vs TV Shows
* Content release trends over the years
* Top countries producing content
* Most popular genres
* Rating distribution

Visualizations were created using Matplotlib to clearly present findings.

---

## 📈 Key Insights

* Movies dominate Netflix content compared to TV Shows
* Significant growth in content after 2015
* USA contributes the highest number of titles
* Drama and Comedy are the most common genres

---

## 🚀 How to Run the Project

1. Clone the repository:

```bash
git clone https://github.com/your-username/netflix-data-analysis.git
```

2. Navigate to the project folder:

```bash
cd netflix-data-analysis
```

3. Install required libraries:

```bash
pip install pandas numpy matplotlib
```

4. Open the notebook:

```bash
jupyter notebook
```
